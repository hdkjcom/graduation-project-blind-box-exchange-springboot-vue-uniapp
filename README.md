## 🧠 项目简介

本项目是一个基于 **SpringBoot + Vue + Uniapp** 开发的技能互换平台小程序，用户可以发布自己的技能或需求，通过系统进行匹配，实现资源互换与价值连接。

### ✨ 核心功能

- 👤 用户注册登录（支持小程序授权）
- 🫧 泡泡发布（技能/需求发布）
- 🔄 智能匹配（标签匹配推荐）
- 💬 在线聊天（基础IM）
- ⭐ 评价系统（信用体系）
- 🛠 后台管理系统（用户/内容管理）

---

## 🛠 技术栈

- 后端：Spring Boot / MyBatis / MySQL  
- 前端：Vue / Uniapp  
- 小程序：微信小程序  
- 中间件：Redis（可选）
