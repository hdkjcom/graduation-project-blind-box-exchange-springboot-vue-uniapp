## 🧠 项目简介

本项目是一个基于 **SpringBoot + Vue + Uniapp** 开发的「盲盒互换平台小程序」，用户可以发布自己的物品形成盲盒，通过平台进行交换，增加趣味性与社交互动。

📌 核心玩法：
- 用户发布盲盒（隐藏内容）
- 随机匹配交换对象
- 打开盲盒获取惊喜
- 形成互动社交关系

👉 项目定位：  
**“盲盒 + 社交 + 交换” 的轻娱乐社交平台（非常适合毕业设计讲故事）**

## ✨ 核心功能

### 👤 用户模块
- 微信授权登录
- 用户信息管理
- 个人盲盒记录

### 🎁 盲盒系统（核心）
- 发布盲盒（隐藏内容）
- 上传图片 / 描述
- 设置交换条件

### 🔄 互换机制（亮点）
- 随机匹配用户
- 双向确认交换
- 状态流转（待交换 / 已完成）

### 💬 社交互动
- 在线聊天
- 用户互动反馈

### 🛠 后台管理系统
- 用户管理
- 内容审核
- 盲盒管理
