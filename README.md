## 🧠 项目简介

本项目是一个基于 **SpringBoot + Vue + Uniapp** 开发的「盲盒互换平台小程序」，用户可以发布自己的物品形成盲盒，通过平台进行交换，增加趣味性与社交互动。

📌 核心玩法：
- 用户发布盲盒（隐藏内容）
- 随机匹配交换对象
- 打开盲盒获取惊喜
- 形成互动社交关系

👉 项目定位：  
**“盲盒 + 社交 + 交换” 的轻娱乐社交平台（非常适合毕业设计讲故事）**
